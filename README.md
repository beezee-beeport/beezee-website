# beezee-website

Marketing site for **BeeZee** — [beezee.co.in](https://beezee.co.in)

BeeZee is an AI-enabled blue-collar job platform, a product of [BeePORT Technology](https://beeport.in).

Single static page (`index.html`) served via GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or serve it:

```bash
npx serve .
```

## Deployment

Pushing to `main` auto-publishes via GitHub Pages. Custom domain: `beezee.co.in` (configured via `CNAME`).
